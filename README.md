# Inventory Management System (MERN Stack)

A simple **Inventory Management System** built using **MongoDB, Express.js, React, and Node.js (MERN)**.  
This system allows user authentication, product management, sales tracking, and viewing sale history.

---

## 🛠 Features

### ✅ Authentication
- User registration & login
- JWT-based authentication
- Password reset via token

### ✅ Product Management
- Add, update, and delete products
- Display product stock in real-time
- Product details: Name, Price, Quantity

### ✅ Sales Module
- Record product sales
- Reduce product stock automatically
- Track sales by date
- Sale details: Product, Quantity, Price, Total, Date

### ✅ Sale History
- View all past sales
- Data populated from MongoDB
- Easy-to-read table with product info and sale totals

### ✅ Dashboard (Optional)
- Total products
- Total sales
- Low stock alerts

---

## 💻 Tech Stack
- **Frontend:** React, Axios, React Router, Tailwind CSS / CSS
- **Backend:** Node.js, Express.js, JWT authentication
- **Database:** MongoDB (using Mongoose)
- **Other:** LocalStorage for storing JWT token
